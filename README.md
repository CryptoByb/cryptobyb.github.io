# CryptoByB — Stablecoin Blacklist Checker

Paste a wallet address and instantly know if it's been blacklisted by USDT or USDC — no signup, no tracking, nothing stored.

**Check now:** https://cryptobyb.pages.dev/

## What you can do

- **Blacklist check** — see if a wallet is frozen on any supported chain
- **Token approvals** — find which contracts can spend your USDT/USDC and revoke them
- **TX Tracer** — follow USDT/USDC transfers between two wallets
- **Batch check** — paste multiple addresses and check them all at once
- **Risk score** — get a single LOW / MEDIUM / HIGH / CRITICAL rating so you know exactly what you're dealing with

## Why trust it

Everything runs directly in your browser. No servers, no accounts, no data collection — just raw on-chain data from public RPC nodes.
